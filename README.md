# Brainstorming

An HTML-only brainstorming board for rapid ideation and project selection.

## Quick Start

Open `index.html` in any modern browser. No build step, no server required.

## Workflow

1. **Brainstorm** — Add cards to columns
2. **Evaluate** — Set priorities, drag between columns
3. **Select** — Mark the winning idea
4. **Archive & Reset** — Export the session and clear the board

## Columns

- Agent Harness
- AI / ML Training
- Arxiv Research
- Tools & Infrastructure
- Experiments

## Archive

Exported sessions are saved to `archive/` as dated HTML files.
