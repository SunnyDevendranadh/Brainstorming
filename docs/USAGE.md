# Usage

## Adding cards

Click `+ Add` on any column. Type the idea, press Enter.

## Moving cards

Drag a card between columns to change category. The order within a column is preserved.

## Setting priority

Click the dot on a card to cycle priority: low → med → high.

## Selecting the winner

Click the star icon. Only one card can be the winner at a time.

## Exporting

Click `Archive & Reset`. The board is saved as a dated HTML file in `archive/` and the live board is cleared.
